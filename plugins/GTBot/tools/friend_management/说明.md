# friend_management

这个插件为 GTBot 提供一个给 LLM 主动调用的好友删除工具：`delete_friend`。

## 配置

文件位置：

`plugins/GTBot/tools/friend_management/config.json`

字段说明：

- `enabled`
  - 是否启用删好友能力
  - 默认是 `false`
- `api_action`
  - OneBot 动作名
  - 默认是 `delete_friend`
- `timeout_sec`
  - 调用超时时间
- `protected_friend_ids`
  - 受保护好友 QQ 号列表
  - 列表中的好友不会被工具删除
- `protected_friend_notes`
  - 受保护好友的备注说明
  - key 使用字符串形式的 QQ 号

示例：

```json
{
  "enabled": true,
  "api_action": "delete_friend",
  "timeout_sec": 15.0,
  "protected_friend_ids": [123456789, 987654321],
  "protected_friend_notes": {
    "123456789": "老板",
    "987654321": "测试保留账号"
  }
}
```

## 工具行为

- 当 `enabled=false` 时，工具会直接拒绝执行
- 当目标 QQ 号在 `protected_friend_ids` 中时，工具会拒绝删除
- 真正执行时，会调用：

```python
bot.call_api(api_action, user_id=target_user_id)
```

## 注意

- 这个插件默认是高风险能力，建议仅在你确认底层 OneBot 实现支持 `delete_friend` 时启用
- 如果你使用的实现动作名不同，可以修改 `api_action`
